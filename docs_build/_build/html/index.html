pyfortuneは某サイトへのスクレイピングを補助するツールです。

インストール¶

pipコマンドを利用してインストールします

$ pip install py-fortune

クイックスタート¶

ログイン処理のみを行う簡単なサンプルは以下の通りです

>>> from pyfortune.session import Session
>>> s = Session()
>>> s.login('username', 'password')
'username'

status 関数を利用することでログイン済みかどうかを確認出来ます。

>>> s.status()
'login'
